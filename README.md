# nine-to-five-pwa
This is a PWA version of the original [nine-to-five](https://github.com/tommiseppanen/nine-to-five) React Native app I wrote in 2018. I was trying out Antigravity to do the conversion. 

# How to run & deploy
To start local version
```
npm run dev
```

To deploy to gh-pages
```
npm run deploy
```
